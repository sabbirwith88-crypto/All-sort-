🎮 HOW TO PLAY

Water Sort Puzzle Pro is a challenging brain teaser!

🎯 Goal: Sort all colors so each tube contains only one type

👆 How to play: Tap a tube to select, then tap another tube to pour

✓ Rules: You can only pour same colors together

🎯 Challenge: Complete each level in minimum moves

Features:

  • 🎨 Beautiful colors with smooth animations
  • 💡 Hint system to help when stuck
  • ↩️ Undo moves if you make a mistake
  • 📊 Track your best scores
  • 🎁 Get special reward code for level 100 completion!

Level Progression:

  • Level 1: 3 tubes (Easy start)
  • Level 2-4: 4 tubes
  • Level 5-7: 5 tubes
  • Level 8+: Progressive difficulty
  • 🏆 Level 100: Special reward code!

Good luck! 💪
